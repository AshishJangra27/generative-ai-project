# AI Content Writer, Bulk Creator & Chatbot (Gemini Projects)

This project demonstrates how to build **3 practical Generative AI applications** using **Google Gemini API**:
1. AI Content Writer  
2. Bulk AI Content Creator  
3. Custom AI Chatbot  

All projects use the **Gemini 2.5 Flash model** to generate fast, cost-efficient, and high-quality responses.

These projects help you understand how real-world GenAI tools like content writers and chatbots actually work behind the scenes.

## What This Project Covers

- Generate high-quality AI articles using Gemini  
- Automate bulk content creation for multiple topics  
- Build a conversational AI chatbot  
- Maintain chat history for contextual responses  
- Track token usage for cost awareness  
- Save AI-generated content as publish-ready files

## Project 1: AI Content Writer

### What it does
Generates a clean, publish-ready article on any topic using Gemini.

### Features
- Accepts a topic as input  
- Writes simplified article (‚â§200 words)  
- Avoids heavy jargon  
- Saves output directly as `.txt` file  
- Ready for blog/website publishing  

### Workflow
1. Provide topic name  
2. Create a prompt with writing instructions  
3. Send prompt to Gemini  
4. Receive article response  
5. Save the article as a text file  

---

## Project 2: Bulk AI Content Creator

### What it does
Automatically generates articles for **multiple topics at once**.

### Features
- Predefined topic list  
- Loops through all topics  
- Generates article for each topic  
- Saves each article separately  
- Useful for SEO/blog automation  

### Workflow
1. Create list of topics  
2. Loop through each topic  
3. Send structured prompt to Gemini  
4. Generate article  
5. Save each file automatically  

---

## Project 3: AI Chatbot (Conversational Bot)

### What it does
Creates a basic conversational chatbot using Gemini with memory.

### Features
- Continuous conversation loop  
- Maintains chat history  
- Context-aware responses  
- Token usage tracking  
- Exit option anytime  

### Workflow
1. User asks a question  
2. Chat history stored  
3. Full context sent to Gemini  
4. Gemini generates an answer  
5. Token usage displayed  
6. Conversation continues
